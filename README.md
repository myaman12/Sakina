
# Sakina v2.8 - Sheikh Qassas Edition (Sakina_28)

**Repository:** [https://github.com/myaman12/Sakina_26](https://github.com/myaman12/Sakina_26)
**Branch:** `sakina_28`

Sakina is a cross-platform dynamic wallpaper application that combines high-quality surreal 4K video streams with adaptive spiritual audio and AI-curated wisdom.

## v2.8.1 Features (Sakina_28)

*   **Randomized Drone Cinematography**: In Quran mode, the app now intelligently selects a random video from the *top 10* highest-rated 4K drone/aerial clips (Nature theme) instead of repeating the same video.
*   **Sheikh Qassas Edition**: Priority audio logic dedicated to the Adhan and recitations of Sheikh Muhammad Marwan Qassas.
*   **Native Audio Player**: Implemented a robust HTML5 Audio fallback for `audio.com` assets to bypass CORS limitations.
*   **Dynamic Visuals**: Integration with Pexels, Pixabay, and Apple Aerials for surreal, high-definition backgrounds.
*   **AI Curation**: Uses Google Gemini Flash 1.5 to contextually select quotes and Duas based on the current visual theme and time of day.
*   **Multi-Language Support**: English, Dutch, and Turkish.

## Technologies

*   React 19
*   TypeScript
*   Tailwind CSS
*   Google GenAI SDK (@google/genai)
*   Lucide React
